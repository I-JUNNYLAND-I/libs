|:--------:|:---:|
| socket() |소켓 생성|
| bind()   |소켓에 주소 할당|
| listen() |연결 요청 대기|
| accept() |연결 요청 수락|
| connect()|연결 요청|
| send()   |데이터 송신|
| recv()   |데이터 수신|
